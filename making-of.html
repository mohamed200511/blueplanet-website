Design impacts in Pages

1- Colors

The pages used various colors types, but it relies heavily on blue and white

  • Dark Blue #2163C5 with White Background: main heading, images and backgrounds in home page
  • Light Blue #0d6efd with White Background: subheading and their descriptions in home page
  • White #ffffff with #2163C5, #0d6efd and Blue Images: both main heading and subheading used in all pages around banners, and navigation bars

those colors used to reflect the importance of ocean conservation. Furthermore, by combining blue and white background we create a visually appealing look across the entire home page and ensure easy readability

in the content page (three sub topics). We divided the page into two colors so that we separate the two pages into references (light gray) content and main content (white). Moreover, we used the black color text in both main heading and their descriptions, Because black text gives the highest contrast on light backgrounds, making information easier to read, clearer, and more accessible for all users. Also we used the gray color in all the pages for their introductions/overview

2- Interaction

Several interactive features are implemented to allow visitors to engage with the website:

  1. The current page in navigation bar is highlighted in white, while inactive links remain gray for easy differentiation.
  2. Buttons in home page such as "Contact Us" and "Sources, Impacts, Solutions Pages" uses hover transitions.
  3. Navigation bar (light gray) and buttons (background white and text black) changes their colors when on hover.

All those features are added to improve the user engagement across all Web pages.

3- Consistency of Layout

Sections are aligned and spaced using Bootstrap's grid system. For example, 4-column layout for icons Home Page, 3-column layout for the menu Home Page, 3-column layout for .sources-info with 9-column for .content Sub topics pages. When viewed on smaller screens, these columns stack smoothly, preserving readability and structure.

Impact of HTML, CSS, JS, and Bootstrap on Maintainability

The Structure of HTML is designed in clean and organized way that makes the website easy to maintain and develop. On the other hand, the CSS ensures visual consistency across the entire website and makes styling changes extremely efficient. Furthermore, using javascript and bootstrap to enhance the interaction and register form.

  • Reusability: If we want to add more HTML pages we can easily reuse the class. For example, we can reuse the footer class across all the pages
  • Unified layout: All the pages follow a consistent layout. For instance, all the pages consist of Navigation then Banner then Content and last is footer. Thus, the developer can quickly update the site and reduce its confusion.
  • Semantic Structure: The HTML and CSS uses clearly named classes that describe their purpose. For example, a class such as .banner immediately tells the developer that this section controls the banner area. This improves readability, reduces the need for documentation, and helps developers understand the function of each part of the page at a glance.
  • External CSS File: this CSS feature uses a one CSS external file that contains all the designs. for example, if we want to change the font color in banner class for the entire web, we can easily refer to stylesheet file and change it. This ensures no time is wasted and easy maintenance.
  • Repeated Use of Functions: JavaScript uses a function to repeat in case the developer wants to use the function again. For example, the developer can use the display function again to display the visitor's name anywhere on a page and on any page they want.
  • Easy Responsive: using the bootstrap rather than the CSS it will give the page responsive feature for every size of device. For example, we use the navbar bootstrap class to give the best responsive. This can help the developers by reducing the size of codes in CSS.
